## 介绍

一个简单php练习项目

### 使用方式

把该项目引入服务器下，

修改install/mysql.php下的数据库连接信息。

执行/install/mysql.php 即创建数据库。

/index.php 为博客首页

/admin/index.php 为后台管理首页

后台管理实现了 博文的增加，删除，修改，查询（查询由js实现）
